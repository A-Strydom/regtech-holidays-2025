## What's This?

This is a fun holiday greeting card web app that brings the RegTech team's festive spirit to life. Watch as the RegTech sleigh soars through the Northern Lights (featuring Cape Town and Joburg) into 2026. Share your goals to smash in the upcoming year!

## How to Use

1. **Start the Journey**: Click on the sleigh or press Enter to begin the animation
2. **Watch the Magic**: Enjoy the sleigh flying through the Northern Lights
3. **Enter a Challenge**: Type something you want to "smash" in 2026
4. **SMASH It**: Watch as your text flies up and gets smashed with confetti!
5. **Reset**: Start over anytime with the Reset button

## Setup

Simply open `index.html` in your browser, or deploy to GitHub Pages for easy sharing!

## Tech Stack

- Pure HTML, CSS, and JavaScript
- Canvas API for confetti animations
- Web Audio API for celebratory sounds
- CSS animations and transforms for smooth performance
